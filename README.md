# Movie Recommendation App

## Project Overview
This project is a Flask-based movie recommendation/search service developed for IS2209.  
It integrates with the OMDb API and a Supabase PostgreSQL database.  
The app allows users to search for movies and store favourite movies in the database.

## Features
- Search for a movie using OMDb API
- Save favourite movies to Supabase/PostgreSQL
- Retrieve saved favourites
- Health check endpoint
- Status endpoint
- Docker support
- GitHub Actions CI
- Live deployment on Render

## Tech Stack
- Python
- Flask
- OMDb API
- Supabase / PostgreSQL
- Git & GitHub
- GitHub Actions
- Docker
- Render
